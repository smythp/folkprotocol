# Symptom-Specific Focus Protocols

## Protocol Selection by Primary Symptom

### ADHD Symptoms (Restlessness, Impulsivity, Hyperactivity)

**Primary Protocol:**
```
Bacopa monnieri 300-600 mg daily (50% bacosides)
+ Ashwagandha 225-400 mg daily
```

**Evidence:**
- Bacopa: 93% reduction in restlessness, 89% improvement in self-control
- Ashwagandha: p<0.001 improvement in ADHD scores (vs placebo p<0.217)
- Synergy: Cognitive enhancement + stress reduction

**Timeline:** 8-12 weeks for Bacopa, 30-56 days for Ashwagandha

**Alternative Options (All "Fair Indication"):**
- Melissa officinalis (lemon balm)
- Matricaria chamomilla (chamomile)
- Valeriana officinalis (valerian)

**Traditional Formula:**
```
Ayurvedic Rasayana blend:
- Brahmi (Bacopa)
- Ashwagandha
- Shankhpushpi
- Gotu Kola
```

## Symptom Severity Protocols

### Mild Symptoms (Occasional brain fog, mild distractibility)

**Start with single herb:**
- L-theanine + Caffeine (acute, as needed)
- American ginseng 100 mg (working memory)
- Rosemary 500 mg (students) or 750 mg (older adults)

**Timeline:** Acute to 1 month

---

### Moderate Symptoms (Consistent focus issues, noticeable memory problems)

**Two-herb combination:**
- Bacopa 300-600 mg + Ashwagandha 225-400 mg
- Rhodiola 170-576 mg + Polygala 300 mg/day
- American ginseng 200-400 mg + L-theanine + Caffeine as needed

**Timeline:** 4-12 weeks

---

### Severe Symptoms (ADHD, significant impairment, multiple domains affected)

**Comprehensive protocol:**
```
Foundation:
- Bacopa monnieri 600 mg daily (cholinergic + memory)
- Ashwagandha 400 mg daily (stress + executive function)

Add symptom-specific:
- Mental fatigue: + Rhodiola 576 mg
- Poor attention: + L-theanine 100mg + Caffeine 50mg (acute)
- Working memory: + American ginseng 200-400 mg

Consider traditional formula:
- Ayurvedic Rasayana (Bacopa + Ashwagandha + Shankhpushpi + Gotu Kola)
- TCM Gui Pi Tang (if anxiety/overthinking component)
```

**Timeline:** Minimum 8-12 weeks for full assessment

## Key Principles

1. **Match herb to symptom mechanism:**
   - Fatigue → Rhodiola (dopamine, anti-fatigue)
   - Memory → Bacopa (acetylcholine, BDNF)
   - Stress fog → Ashwagandha (cortisol reduction)
   - Attention → L-theanine+Caffeine (synergistic alpha modulation)

2. **Acute vs chronic:**
   - Acute needs: L-theanine+Caffeine, American ginseng low dose
   - Chronic needs: Bacopa, Polygala, Rhodiola (4-12 weeks)

3. **Single herb vs combination:**
   - Mild symptoms: Single herb adequate
   - Moderate-severe: Combinations outperform single herbs

4. **Traditional formulas provide time-tested combinations:**
   - Ayurvedic Rasayanas for holistic brain support
   - TCM Gui Pi Tang for mental fatigue from overthinking
   - TCM Bu Nao Wan for memory/kidney deficiency

5. **Age considerations:**
   - Younger adults: Higher doses often tolerated
   - Elderly: Lower doses optimal (Rosemary 750mg vs 500mg)
   - Children: ADHD protocols require professional guidance
