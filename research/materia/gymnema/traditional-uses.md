# Traditional Uses - Gymnema sylvestre

## Note on Bibliography Coverage

**Important**: The current bibliography (bibliographies/gymnema.yaml) contains exclusively modern clinical research (systematic reviews, meta-analyses, and randomized controlled trials from 1990-2024). It does not include sources on traditional uses, historical applications, or ethnobotanical documentation.

## Known Traditional Context (Not Documented in Current Bibliography)

Gymnema sylvestre is known to have traditional use in:
- **Ayurvedic medicine** (India) - commonly referenced in modern sources
- Sanskrit name: "Gurmar" (literally "sugar destroyer")

However, **no specific traditional use sources with dates, preparation methods, or historical documentation are included in the current bibliography.**

## What Would Be Needed

To properly document traditional uses, the bibliography would need sources covering:

1. **Historical texts**: Ancient Ayurvedic texts (with dates/centuries)
2. **Ethnobotanical surveys**: Documentation of traditional preparation methods
3. **Traditional preparations**:
   - Water extracts/decoctions
   - Leaf chewing practices
   - Traditional dosing regimens
   - Regional variations in use
4. **Traditional indications**: Beyond diabetes (digestive issues, etc.)
5. **Cultural context**: Which communities/regions used it, when, and how

## Recommendation

If traditional uses documentation is needed for the materia page, additional sources should be added to the bibliography focusing on:
- Ayurvedic historical texts and commentaries
- Ethnobotanical field studies
- Traditional medicine documentation from India and other regions where Gymnema grows

The current bibliography provides excellent clinical evidence but lacks the historical/traditional dimension.
